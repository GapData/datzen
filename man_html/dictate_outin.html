dictate_outinR Documentation

The dictate_outin() function

Description

An extremely simple function that creates a named vector. A useful helper to create key-value dictionaries to use in str_replace_all

Usage

dictate_outin(swap_out, swap_in)

Arguments

swap_out

vector whose elements will be used as the names of the resulting character vector

swap_in

vector whose elements will be used as the values of the resulting character vector

Value

a character vector with values from swap_in and names from swap_out

See Also

str_replace_all

Examples

dictate_outin('compton','bompton')
