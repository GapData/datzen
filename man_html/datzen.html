datzen: A package containing miscellaneous helper functions to achieve Data Zen.

Description

The datzen package provides three categories of commonly used Zen functions: Themed Branding, Special Computations, and Data In/Out. The goal of this package is to help data analysts rapidly prototype common tasks.
