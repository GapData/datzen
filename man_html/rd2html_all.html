rd2html_allR Documentation

The rd2html_all() function to batch convert rd files to html files

Description

uses the tools::Rd2HTML function inside base::lapply

Usage

rd2html_all(dir_man_rd, dir_man_html)

Arguments

dir_man_rd

a character string for the directory containing .rd files

dir_man_html

a character string for the directory to output .html files

Value

nothing is returned, but external .html files are written to dir_man_html
